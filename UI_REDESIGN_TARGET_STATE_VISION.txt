================================================================================
TELLURIDE SKI HOTELS - TARGET STATE VISION
================================================================================
Document Created: November 9, 2025
Purpose: Exhaustive description of ideal end-state design for all pages
Vision: Professional, industry-standard hotel booking website that rivals
        Booking.com, Expedia, and Airbnb in quality while maintaining unique
        Telluride ski resort character

================================================================================
DESIGN PHILOSOPHY & PRINCIPLES
================================================================================

CORE PRINCIPLES:
1. PROFESSIONAL FIRST
   - NO emojis anywhere on the site
   - Professional Lucide React icons exclusively
   - Clean, modern aesthetic matching industry leaders
   - Trust-building design elements throughout

2. IMAGE-RICH STORYTELLING
   - Extensive use of media library throughout site
   - Hero sections with high-quality background images
   - Split layouts combining images with content
   - Visual storytelling that sells Telluride experience

3. FUNCTIONAL EXCELLENCE
   - Every interactive element works flawlessly
   - Smart filtering and sorting
   - Real-time data displayed beautifully
   - Smooth animations and transitions
   
4. CONSISTENT DESIGN SYSTEM
   - Unified component library
   - Consistent spacing, typography, colors
   - Reusable patterns across pages
   - Predictable user experience

5. CONVERSION-OPTIMIZED
   - Clear calls-to-action
   - Strategic internal linking
   - Reduced friction in booking flow
   - Trust signals throughout

================================================================================
UNIFIED DESIGN SYSTEM SPECIFICATIONS
================================================================================

ICON SYSTEM:
Library: Lucide React (already installed)
Usage: EXCLUSIVE - no inline SVGs, no emojis, no other icon libraries
Size Standards:
  - Small (UI elements): 16px (w-4 h-4)
  - Medium (features): 20-24px (w-5 h-5 or w-6 h-6)
  - Large (hero features): 32-40px (w-8 h-8 or w-10 h-10)
Color: Semantic based on context (primary-600, sky-600, etc.)

TYPOGRAPHY SCALE:
Font Family: Poppins (already configured)
Headings:
  - Display XL: 72px / bold / -0.02em tracking (hero headlines)
  - Display LG: 56px / bold / -0.02em tracking
  - Display MD: 48px / bold / -0.02em tracking
  - Display SM: 36px / bold / -0.01em tracking
  - H2: 30px / semibold
  - H3: 24px / semibold
  - H4: 20px / semibold
Body:
  - Large: 20px / 1.6 line-height
  - Regular: 16px / 1.5 line-height
  - Small: 14px / 1.5 line-height
  - Tiny: 12px / 1.4 line-height

SPACING SYSTEM:
Following Tailwind defaults but emphasize:
  - Section padding: py-16 lg:py-24 (generous whitespace)
  - Card padding: p-6 lg:p-8
  - Component gaps: gap-6 lg:gap-8
  - Grid gaps: gap-6 for cards, gap-8 for sections

CARD SYSTEM:
Base Card:
  - Background: white (bg-white)
  - Border: border border-neutral-200
  - Radius: rounded-2xl
  - Shadow: shadow-card
  - Hover: shadow-card-hover hover:-translate-y-1 transition-all duration-300
  - Padding: p-6 lg:p-8

Product Card (Hotels/Activities):
  - Equal height in grids (use min-h or flex)
  - Image: aspect-[4/3] object-cover rounded-t-2xl
  - Content: p-6 flex flex-col
  - Footer (price/CTA): mt-auto (always at bottom)
  - Hover effect: scale image slightly, lift card

BUTTON SYSTEM:
Primary CTA:
  - bg-primary-600 hover:bg-primary-700
  - text-white font-bold
  - px-8 py-4 (large), px-6 py-3 (medium), px-4 py-2 (small)
  - rounded-xl
  - shadow-card hover:shadow-card-hover
  - hover:scale-105 transition-all duration-300

Secondary CTA:
  - bg-white text-primary-600
  - border-2 border-primary-600
  - Same sizing/spacing as primary
  - hover:bg-primary-50

Icon Button:
  - p-2 or p-3
  - rounded-lg
  - hover:bg-neutral-100
