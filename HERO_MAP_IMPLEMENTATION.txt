INTERACTIVE HERO MAP + SEARCH IMPLEMENTATION
============================================

WHAT WAS BUILT
--------------
Interactive hero section combining full-screen Mapbox map with search functionality.

FILES
-----
Created:  src/components/lodging/HeroMapSearch.tsx
Modified: src/pages/index.astro

FEATURES
--------
- Full-screen interactive Mapbox GL map with hotel markers
- Featured hotels highlighted with amber pulsing markers (top-rated)
- Click markers for hotel popups with details/prices
- Date picker (check-in/check-out) and guest selector
- Three map styles: Streets, Satellite, Terrain
- Glassmorphism search panel overlay
- Legend below map explaining marker colors
- Responsive: desktop (expanded) / mobile (collapsible)
- Smooth animations and fly-to interactions

TECHNICAL
---------
Stack: React, TypeScript, Mapbox GL, react-map-gl, Tailwind CSS
Dependencies: All already installed (mapbox-gl, react-map-gl, lucide-react, date-fns)
Bundle: +230KB (Mapbox + wrapper)

USAGE
-----
Component automatically integrated in homepage (src/pages/index.astro):

<HeroMapSearch 
  client:load 
  initialHotels={allHotels} 
  minPrices={minPrices}
  currency="USD"
/>

Props:
- initialHotels: LiteAPIHotel[] - Hotels with location.latitude/longitude
- featuredHotelIds: string[] - IDs of featured hotels (shown with amber pulsing markers)
- minPrices: Record<string, number> - Optional price mapping
- currency: string - Price currency code

MARKER COLORS:
- Amber (pulsing): Featured/top-rated hotels
- Blue: Regular hotels
- Orange: Hovered hotel
- Red: Selected hotel

CONFIGURATION
-------------
Environment: Requires MAPBOX_TOKEN
Map Center: TELLURIDE_CENTER in src/lib/mapbox-utils.ts
